# Qianqian (Bella) He — Portfolio Site

This GitHub Pages site is refactored for industry-facing roles: Machine Learning Engineer, AI Engineer, Data Scientist, Software Engineer, ML Researcher, and Product Manager.

## Update instructions

1. Copy `index.html`, `_config.yml`, and the `assets/` folder into the root of `belsguidetotech.github.io`.
2. Commit and push to the `main` branch.
3. GitHub Pages will rebuild the site automatically.

```bash
git add index.html _config.yml README.md assets/QianqianHe_Industry_Resume.pdf
git commit -m "Refactor portfolio for industry AI/ML roles"
git push origin main
```
